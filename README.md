## How to use

**설치방법**

1.create-react-app 설치

```
create-react-app
```

2.yarn 설치

```
npm install --global yarn
```

3.실행하기

```
yarn start
```

**_배포 방법_**
1.yarn build

```
yarn build
```

2.deploy netilfy

```
deploy netlify
```
